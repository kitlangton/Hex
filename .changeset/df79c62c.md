Coach v3: streaming UX, popover position, and reliable rewrite.

- Model now produces Markdown instead of JSON — the streaming view renders it live with MarkdownUI as headers, bullets, and emphasis materialize, instead of showing raw JSON characters.
- After the stream completes, the response is regex-parsed into the structured Feedback struct so the polished score-ring / issue-cards view replaces the streaming panel.
- Coach popover replaced with an NSPanel positioned deterministically at the top-right of the active screen with vibrancy background; no longer depends on the menu-bar icon being visible (which it isn't when hidden behind the notch). Click outside dismisses; doesn't activate the app.
- Native phrasing section now always renders, with an "Already native-level" badge when the rewrite matches the original. Prompt updated to require a non-empty rewrite (repeat the original verbatim when no improvement is possible) so the field is reliably populated.
