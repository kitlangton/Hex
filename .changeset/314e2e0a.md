Coach prompt is now fully user-editable.

- Settings → Pronunciation Coach → "Coach prompt" exposes the entire prompt template as a monospaced TextEditor, with a "Reset to default" button when customized. The default is shipped as a constant so the editor always shows the same baseline.
- The default template uses placeholders: `{{TRANSCRIPT}}`, `{{L1_LANGUAGE}}`, `{{TARGET_ACCENT}}`, `{{USER_GOAL}}` / `{{USER_GOAL_BLOCK}}`, `{{CUSTOM_GUIDANCE}}` / `{{CUSTOM_GUIDANCE_BLOCK}}`. Custom templates may use any subset; if `{{TRANSCRIPT}}` is omitted the transcript is appended at the end automatically.
- `Feedback.rawMarkdown` now stores the model's full response verbatim. When a custom prompt produces output that doesn't match the structured-card parser (Score / Summary / Native phrasing / Issues / Wins), the popover falls back to rendering the response as Markdown verbatim — tables, headers, blockquotes and all. The default prompt continues to produce the card view.
