Coach now streams feedback live: each analysis hits Gemini's `streamGenerateContent` SSE endpoint, the popover window automatically pops open as soon as analysis starts (toggleable in Settings as "Automatically open coach window"), and the raw model response renders in real time as it arrives. When the JSON parses, the popover swaps to the structured view. The feedback schema gains a `native_rewrite` field — the model rewrites the user's transcript the way a native speaker would phrase it, shown alongside the original with a one-click Copy button. A new Settings field, "Custom guidance", lets users append free-form context to the coach prompt (e.g. "I'm a software engineer — lenient on jargon") without exposing the underlying schema.
