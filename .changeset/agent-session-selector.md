Add an agent selector to the Agent Plugins voice window. When several Claude sessions have been seen this run, the header shows a row of project avatars — tap one to retarget the window to that session (its live card if it's blocked on a hook, or a fresh compose card if it's idle). Summoning now lands on whatever you were last working with, with the selector to switch from there. When there's nothing to target — no remembered session and no running Claude terminal — the window shows a clear "No Claude sessions" message instead of opening a compose box that would have typed into a random frontmost window.
