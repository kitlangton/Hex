Scaffold HexCoach M1 plumbing: CoachFeature reducer observes the shared transcription history and dispatches a Gemini pronunciation analysis for recordings longer than the user-configured threshold, with results stored in a new coach feedback file. Adds a KeychainClient for BYOK API key storage, a PronunciationProvider protocol with Gemini live and OpenAI stub implementations, a nested CoachSettings struct on HexSettings (coach defaults to off), and the `com.apple.security.keychain` entitlement. No UI yet — coach runs silently from logs.
